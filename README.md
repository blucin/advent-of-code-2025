# Advent of Code 2025 - Go solutions

This repo contains my solutions for the [Advent of Code 2025](https://adventofcode.com/2025) challenges.

> [Advent of Code 2025](https://adventofcode.com/2025) is an Advent calendar of small programming puzzles for a variety of skill levels that can be solved in any programming language you like. People use them as interview prep, company training, university coursework, practice problems, a speed contest, or to challenge each other.

## How to run

1. Make sure you have Go installed.
2. Clone this repository.
3. From the project root:

```bash
go run .\cmd\aoc\main.go <day_number>
```
